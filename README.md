# Finsage Platform

An educational web-based platform for novice digital-asset investors.

## Planned stack
- Frontend: React + TypeScript + Vite
- Backend: FastAPI + Python
- Database: PostgreSQL
- Auth: JWT authentication
- Simulation: Monte Carlo
- AI/ML: Scam detection classifier
- Reporting: PDF export

## Project focus
This project is designed to help student and beginner investors understand digital asset investing through:
- conversational onboarding
- explainable risk profiling
- portfolio scenario generation
- Monte Carlo simulation
- scam detection and risk warnings
- dashboard reporting

This platform is educational only and does not provide financial advice or execute real trades.
